How it Runs

As told before it  runs with JavaScript , in which the simple snippet converts the characters like < to '&lt; , < to &gt; and much more characters can be parsed like this.Now lets move on to JavaScript Code part
function $(esc_tool){ 
return document.getElementById(esc_tool) 
} 
var char2entity = { "'" : '&#39;', '"' : '&quot;',  '<' : '&lt;', '>' : '&gt;',  '&' : '&amp;'};
function escape_entities(str) { 
var rv = ''; 
for (var i = 0;i < str.length; i++) { 
var ch = str.charAt(i); 
rv += char2entity[ch] || ch; 
} 
return rv; 
} 
function do_escape(e){ 
$('escaped').value = escape_entities(e.value) 
}

Detailed Tutorial for Replacing the Character

This tutorial may be a bit geeky for those who are unfamiliar with JavaScript Coding, So you can Skip this part.
Here as mentioned above you can increase the list of characters by adding the character to escaped with the character to be replaced instead in this format.
Consider ## and ** to be two characters.The format to be followed is -

' ## ' : ' ** ' ,

Here
  • Replace ## with the character to be excluded
  • Replace **  with the character to be replacing the previous ## character
  • a Colon (:) must be there after ##  followed by ** (and if you have a big list you should not forget to add a Comma(,)
  • Result is that ## becomes **
Here is where you should edit .
var char2entity = {  '##' : ' ** ', PUT HERE YOUR LIST'};
