Adding Custom fonts to your blogs and sites have become a trend nowadays.I had already written about adding custom fonts to blogger earlier and it had recieved pretty good hits.Everyone(including me)was consious about the the time lag created by the custom fonts .As you know almost all the browsers (eg IE,chrome ) will not load anything until your custom font is loaded.That creates a huge time lag.
Today we will be dealing with this problem and fighting this .
Here we will be using Jquery script which was made by Omar AL Zabir (@omaralzabir) and edited by me for using it in blogger.

The main objective of this script is it pre caches the  font-file based on browser version and loads them before loading the CSS file.And with the help of Jquery we immediately loads the font file and thus it gets cached down there in the local disk and wont be loaded again.

Getting it done

Things to do before using this script (Read them here on my another post)
Create CSS file for font
Choose or Create Web Based fonts

Using the Script 

Find the </head> and paste the below code snippet above it.



/*Pre Caching font file script for Blogger inspired from Omar AL Zabir (@omaralzabir) */
var fontFilelink = jQuery.browser.msie ? 'url_of_Your_Font.eot'
    : 'url_of_Your_Font.otf'; /*Checking whether it is IE loads font for it*/
var cssFilelink = jQuery.browser.msie ? 'For_IE_url_of_Your_Font.css'
    : 'Not_IE_url_of_Your_Font.css';/*Checking whether it is IE loads CSS for it*/
 /*Begining the Jquery function*/
jQuery(document).ready(function(){
    jQuery.ajax({
    /*Font file Jquery function*/
      url: fontFilelink,
      beforeSend: function ( xhr ) {
        xhr.overrideMimeType("application/octet-stream");
      }
    }).done(function (data) {
        jQuery("    <link>")
          .appendTo(jQuery('head'))/*Appends to <head> tag*/
          .attr({type : 'text/css', rel : 'stylesheet'})
          .attr('href', cssFilelink);
    });});
Now Replace
  • Url_of_Your_Font.eot with your fonts .EOT file
  • Url_of_Your_Font.otf  with your fonts .OTF file
  • For_IE_url_of_Your_Font.css with your CSS for IE
  • Not_IE_url_of_Your_Font.css  with your CSS for other browsers


Remove the first line if you have already included jquery in your template.
Save your template

Thats all you are now set to rock your page
