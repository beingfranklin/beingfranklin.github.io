Dropbox is a wonderful online cloud service which helps you to store your data files on their servers.For Basic users (free users )they gives space up to 18 GB (2 GB+ referral space)and for Paid plans even more.Its so easy to use ,either you can install their software or you can use it online.
Today we are going to learn how to host CSS files using Dropbox
. But Before that ..

Why Dropbox is preferred to host CSS externally

Update : its no longer preferable

Even though many services like Google code offers free service and it can be used to render JavaScript files.But in case of CSS ,it is not a better idea as the file is not delivered accurately.And we need it(CSS) to be delivered badly as it affects our whole design of our blog. Apart from Google code ,the best reliable service is offered by Dropbox.Thus we use it :)

And the most important reason is ,as per browserdiet it helps in browser caching which will increase* the speed of your website.We checked its loading time and found that it may take few milliseconds more during the page load ,But you can rely on it .

*Note - It works only if the user has enabled Cache in his/her browser
Test your blog using Pingdom tools

Advantages of Hosting CSS files in Dropbox

  • 99% Server Uptime
  • Good file Delivery system
  • Good for less traffic Blogs

Disadvantages of Hosting CSS files in Dropbox

  • Of course Dropbox has certain bandwidth limit. As per Dropbox support, it is 20GB/day for Free(Basic) accounts and 200GB/day for Paid (Pro and Teams) . Links which uses more bandwidth will be disabled
  • Lack in Delivery Speed due to more traffic

Hosting Your Blogger CSS Using Dropbox

  1. Go to Template -> Edit Html 
  2. Find all the CSS in between <b:skin><![CDATA[  and  ]]></b:skin>
  3. Cut and Paste it new notepad file and Save it .
  4. Rename it to Anyfilename.css
  5. Go to Dropbox
  6. Upload  it dropbox 's Public folder
  7. Copy the public link
  8. Now paste the following code before </head> 
    9. <link href='YOUR PUBLIC LINK' rel='stylesheet'/>
  9. Change YOUR PUBLIC LINK to your link.
  10. Save your template.You are done !!
