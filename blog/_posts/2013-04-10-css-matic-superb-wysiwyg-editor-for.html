 It was long time ago where the developers had to code each and every code solely by hand .Now the time changed everyone are relied upon tools that really helps them a lot . These developer tools and WYSIWYG (“What You See Is What You Get”) Editors have really rocked .Personally these “What You See Is What You Get” tools have really helped me a lot.
Today we are going to take a look at one of the such WYSIWYG Editor.The most important highlight of this WYSIWYG Editor is it is ,it is made for CSS effects which gives our websites a fresh look.This wonderful tool is known as CSS MATIC. It helps you to create different Gradients, varied Frames,beautiful Drop Shadows and Noise Textures.



Who is  behind CSS Matic ?

It is from the Spain,from the makers of Freepik.com and Thumbr.it.Our readers may or may not know Freepik.com. It is the search-engine for free stuffs related to design .It is a must use resource page which gives you relevant information to complete your design project with stunning looks.Since Freepik.com doesn’t host any of these resources changes can occur within the availability of these free Stuffs.

CSS Matic - The Ultimate CSS Resource


Even though if I call it The Ultimate CSS Resource , CSS Matic is not such a thing.You may be thinking  What is this man saying  ? Well actually this WYSIWYG Editor helps us only for four things -for Frames, Gradients,  Textures and Drop Shadows.

Gradient Generator


The Gradient Generator really comes ahead of every tools in these sector.This high quality feature rich tool helps you to create gradients which we used to create using some Editing softwares like  Photoshop , Snag it etc. One can create  gradients completely from scratch  or from given samples. These Gradients which we create can include two or more colours.It can be created with ease and you must try it out to discover its more features.
Hey  I've Created one Simple Gradient using CSS Matic Check it out

Border Radius



This tools allow us to create the rounded borders using css .Every edge can be individually defined or all edges at a time .Like all the tools in CSS Matic the CSS code will change (live) as you change the dimensions.

Noise Texture


Using this tool, you can visually create a textures and can be exported as a PNG +CSS.The CSS codes will be defined as The color of the background, the color of the noise, transparencies and density – all can be as comfortably defined as such to use it with the created PNG.

Box Shadow


This tool helps you to create beautiful drop shadows in CSS.You just need to specify how your shadow should be displayed using the options given,all in all it helps you create stunning effects !!
All in all this is a tool which we would recommend you to integrate into your "daily dose of CSS".This helps you get better speed and workflow while handling the styling part of your works.
