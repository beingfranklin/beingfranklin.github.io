Visual Studio Code , Microsoft's IDE , is now available on Ubuntu as a Snap. According to David Callé the Git integration comes very handy for the developers.

Here it comes with powerful built-in support for JS, Node.js and TypeScript. It also has rich environment support for other languages like C. C# ,PHP, GoLang etc.

Even though its not powerful as its predecessor Visual Studio, we can do very much in this VS Code. But you can customize it by installing plugins. It has wide variety of plugins.


Visual Studio Code in Ubuntu

Is it the only way to install VS Code ?

No. It was possible to install it using Ubuntu umake ( previously Ubuntu Developer Tools Center ) and like installing  usual .deb file. (The links for that are in the references section)
Its now possible to install ubuntu as a Snap.


What is the Big Deal?

Well the greatest advantage is that  by installing VS Code using snap you need not worry about updating the VS Code every time. To be more specific, the snaps autom
atically gets updated or refreshed with the latest snap. So you will be always having the latest VS Code running in your machine.

How To Install VS Code using Snap?



Installing VS Code using Snap


We can install it with simple command 

sudo snap install --classic vscode


P.S if you are installing snap packages for the first time please install snapd shown like below. This service runs in your machine and tracks the installed snaps in your machine.

sudo apt install snapd


Voila ! Your are ready to go.
