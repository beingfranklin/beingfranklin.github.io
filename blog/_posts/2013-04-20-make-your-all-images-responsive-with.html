It Has been a responsive web design revolution where every kind of stuff from text to videos are easily accessible without maintaining  different mobile themes.One of the important one is - Responsive Images Using Commands .That is,Their job is to serve the right image for the occasion.There are many techniques used for responsive images now . We will be going through them in brief and we will learn how to implement these.



Many Ways, But not a Complete Solution

There are many ways to do this. Usual way is by defining the  size and separate location / src  from your server.
<img src="200x100.png" data-1x="400x200.jpg" data-2x="800x400.jpg">
We know It is not at all possible to go back to all the posts or pages and add the data attribute and re-upload different images them when there are lots of posts and pages .This created the need for plug-ins which make the work easier.

Then Sencha.IO  was introduced ,but it too requires the prefixing for the required width or height.
<img src='http://src.sencha.io/320/200/http://domain.com/img/u.png'>
This also was not a complete solution .Even the Photos hosted in  Picasa needed prefixes.
<img src=http://3.bp.blogspot.com/-1iYBZqQq9RM/UVw5txMeIxI/AAAAAAAABjY/WvGn2iKRrvI/s320/alexa-ranking.jpg">
Here we need to Change s320 with other prefixes for resizing .Read more about it by Harish here.

Better Solution : Responsive img Jquery Plug-in

Responsive Img Plugin
By going through many of the plug-ins around the web , the most easiest to use plug-in I found was Responsive img JQuery plug-in.This plug-in resizes the Image according to the device display scale and resizes accordingly.Of course there are many plug-ins out there in the town.  
Apart from others the highlight is that these images gets saved in your servers.Once these images are saved the plug-in delivers from the servers automatically and it is not needed to be created again.

Adding this Plug-in to your site

It is also very easy as 1 2 3 to implement. Just follow these steps.
# Step 0 - Always take backup of your site before you start playing with it.
# Step 1 - Make sure that you have added the Jquery  Plugin to your </head> .If not add it 
<script src='//ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js'></script>
# Step 2 - Download the file and
# Step 3 - Upload it your server and find its URL / location .
# Step 4 - Add the following code above </head> and replace the URL with yours (which you got in # Step 3)
<script async='async' src='your-domain.com/files/js/responsiveImg.js'></script>
Don't worry to add it to the header section as it will not obstruct the other elements as it will be downloaded parallelaly with help of async='async'   (And the file is pretty small too !! )
# Step 5 - Include the following line of JavaScript in your Website (preferably above </head>)
<script> $("img").responsiveImg();</script>
You are done !!!
