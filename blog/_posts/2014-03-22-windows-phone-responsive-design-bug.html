Windows phone had created a boom in the whole world with its awesome metro style.It was a revolutionary move int the segment which shook the world around.Now each and every Designer is minimalist as seen in the windows Phones.

The default browser used ,the IE ,is not as flexible as its other companions . As always IE is far behind in the battle.

There was a bug reported on the responsive design and IE compatibility on the windows phone(384px x 640px)(which according to Microsoft officials,would be corrected soon).And many fixes could be found on the Internet .So I decided to bloggerrise the fix so all of you could use them .

Lets Begin

Step 0  Back up your template (Its very Important !!!). 

Step 1  Got Blogger >> Your Blog >> Template >> Edit HTML 

Step 2  Find <head> and Paste the Code Below.

CSS Part   
<style> 
@-webkit-viewport   { width: device-width; }
@-moz-viewport      { width: device-width; }
@-ms-viewport       { width: device-width; }
@-o-viewport        { width: device-width; }
@viewport           { width: device-width; }
 </style>

Javascript  
<script type='text/javascript'>//<![CDATA[
//Fix Windows phone Bug in Blogger
if (navigator.userAgent.match(/IEMobile\/10\.0/))
//Matching Version
 {
  var msViewportStyle = document.createElement("style");
  msViewportStyle.appendChild(
    document.createTextNode(
      "@-ms-viewport{width:auto!important}"
    )
  );
//View port Width set as Device width  document.getElementsByTagName("head")[0].
    appendChild(msViewportStyle);
}
//End of Fix Windows phone Bug in Blogger .tnx 2 @andreasnorman and
//]]></script>
Step 2   Save Template And You are done!!!
