Chocolatey is a simple automated software management tool which can be controlled by command lines in Windows.

I found it very useful and thought to share it.If you want to know more about it ,please go to their website.

I will be directly jumping to tutorials now.

Installing chocolatey(Recommended method)


Always "Run as Adminidtrator"


  • Open  cmd.exe (here i will be using Command prompt)
    P.S :Always run this as administrator as it requires permissions to install softwares

Installing Chocolatey using cmd.exe

  • Run the below code

@powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))" && SET PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin

  • For Windows Powershell Users ,run th below code

iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))

Success!!

 Installing , Uninstalling and Upgrading Softwares

We can install,uninstall and upgrade softwares very easily with chocolatey short command lines

  • For installing any program Your command goes like this
    choco install <your program>
  • For Uninstalling any program Your command goes like this
    choco uninstall <your program>
  • For Upgrading any program Your command goes like this
    choco upgrade <your program>
