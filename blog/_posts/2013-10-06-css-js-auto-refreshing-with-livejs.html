Today I would like to share a fantastic tool that I had come across lately; The Live.js,plugin created by Martin Kool.
The main function of this JScript file is that it automatically clears and checks for changes in your current CSS and JavaScript files that you are working on , and refreshes them as soon as you finish editing it.
In short , you can open a text editor in one half  and your site in web browser in the other half of the screen. When you save changes in the editor, the updates are reflected immediately in your browser. It really helps to speed up your development time. 


Known Bugs 
A lot of editors offer us live HTML preview, but the major drawback is that they work only  for simple .html page.Here is what is great about Live.js , it works on any kind of webpage, which includes a working WP theme.

Live.js for WordPress Plugin
Here this Idea for Live.js WP Plugin was put forward by noupe.com and all credits goes to them !!

They have combined a  plugin for WP to make it all easy. Here is how you’re gonna unlock this magical tool . 
1.      Download plugin from here 
2.      Affer you unzip it, Rename folder to “pbd-livejs” or even you can see it on Github.com.
3.      Make sure that you are signed in and working locally.And Live.js will be automatically added to your current theme.

Always Remember-Never use this script on a live site as it can make it faulty .It is also adviced to keep an Uptodate Back Up.There is no need to upload this plugin to server with other essential files as it is purely made for developmental process. 

The webpage “flashes” as it loads every time –You may have enabled the  function for sensing  HTML changes.
