Adding Custom Fonts for Blogger
Custom fonts are vastly available which can be added to a site to make it more unique .There are numerous methods to embed fonts to a blog/ website ,from which several of them are paid services like Typekit & Google Webfonts.With the help of CSS (Cascading Style Sheets) you can easily add custom font to your blog/website.This is made possible with @font-face expression.


Today we will be going through different Typography(font styling) and How to implement those on your blogger blogs.

Things to Know Before You Start

First of all lets go through the different types of fonts that are widely used today.They are as follows
    TTF For almost all browsers except Internet Explorer and iPhone Browser
    EOT For Internet Explorer only
    SVG For Apple Products like  iPhone/iPod/iPad.
    WOFF Normally Compressed type ,for emerging standards
By Embedding fonts in these formats would make your custom font visible clearly without the Flash of Unstyled Text(FOUT) issue.FOUT issue happens when our fonts are not cross browser compatible or even happens when the users Internet connection is too slow.Don't worry too much about it now as we will be solving this problem later in this post .

Why to Use Your own Custom Fonts in Blogger

Note : If You are using Wordpress Blogs Read This Tutorial by Matt Russell.

We Know that Blogger Provides Custom fonts in the Template Designer,but those are default fonts(Yes Of course it is beautiful )and all who use Blogger can add it too.So to make our Blog bit more unique we need a different style of Typography for your blogger blog .This will take your blog to a step higher.

Step 0. Backup Your Template
It is very important to back up your template .This would help you to Undo changes even if anything goes wrong.
Step 1. Select or Download A Font From The Collection 

This step (Selecting  Fonts )must be taken with due Care as fonts can be copyright protected.Please do check twice whether they are having commercial / Personal licences.Normally Commercial licenses are a must to use those fonts in your Blogs.

Here are some links where you can find hundreds of fonts.
  • Dafont
  • Fontsquirrel
  • Urbanfonts
Download it .Most probably it will be in OTF or in TTF format.

Step 2. Convert The Downloaded Font to @Font-Face Kit
Convert & Download Your Font Kit
As we know the main step is to convert the downloaded fonts to different formats with @font-face kit.You just need to upload it to one of the following services and which they will convert it into different formats.After conversion you can download the kit.
  • Fontsquirrels(Recommended)
  • Font2web
Note that the kit will contain 4 formats of fonts(inside the fonts folder) , a CSS stylesheet & an index.html file.


Step 3. Upload fonts to the server / Dropbox 
Upload the fonts to Server
In this step we will be uploading the fonts (4 formats) to server.It is necessary to upload to a server or else others wont have access to view it.Since we are using Blogger we do not have access to Google Servers .So we need to upload it your own server(or cloud services).I prefer  Dropbox as it is more reliable and User friendly.(Of course you can use other services too ,But remember that we need the public link with extension ,say your_font.ttf which you can easily obtain from dropbox)

If you are going to use Dropbox, just upload it to Public folder(Link) .You need to copy the Public URL's of all 4 fonts and copy it to a place ,say a notepad file as we need it in our next step .


Step 4. Edit Source Location -
You  need to Change URL of all 4 fonts (Click to Enlarge )

You need to alter all the Source location Url  in the CSS  to the location where you have uploaded it.You need to edit the .woff,.ttf,.svg &.otf source location with your Public Links.

Step 5.Add that Edited CSS Code to Blogger template 

You can add that CSS code by going to the Blogger->Template->Edit Html and Add CSS Code just above ]]></b:skin> and Save it.

 Step 6. Declare the Custom Font using CSS -

In this part you want to apply the Fonts to wherever you want .Most preferred places many people use custom fonts are in the titles and post body .

Post Title - For Using these font in Post Title  Edit  your_font name with your custom font name  and   Add the edited CSS Code just above ]]></b:skin> and Save it.
.post-title {
 font-family: Your_font; /* no need of Exensions like .svg , .ttf ,.otf */
}
Post Body - For Using these font in Post Body Edit your_font name with your custom font name and  Add the edited CSS Code just above ]]></b:skin> and Save it.
.post-body,.post {
 font-family: Your_font; /* no need of Exensions like .svg , .ttf ,.otf */
}
Step 7. Define Back-Up Fonts (Optional but Recommended to avoid FOUT) 

We create a simple back up for Your custom with other fonts so that even if one has slow connection and if the font doesn't load ,He/ She can see the Back up of the fonts.

It is very Easy to add.You just need to extend your font-family declaration with some fonts and comas..You can see the example here.

Post Title
.post-title {
font-family: 'Your_font',Verdana,sans-serif; /* Here also Verdana,sans-serif are added as Back up */
}
Post Body
.post-body,.post {
font-family: 'Your_font',Verdana,sans-serif; /* Here Verdana,sans-serif are added as Back up */
}
