Hello Everyone !!
In this short post we will be showing How to add Schema.Org Mark up to your Blogger blogs with ease.

What is Schema?

Its a kind of micro data that is used by Search Engines to get or interpret the details that is present in a particular website or blog more effectively.

How to Use it?

Note :There are many guides regarding the Schema.org micro data. Therefore we wont be dealing with it .If you want such guides please find it in the resources part.
We are skipping to implementation part directly .

How to Add it to blogger ?

For Post Titles

Find  <data:post.title/> and replace it with


<div itemprop='name headline'>


<data:post.title/>


</div>

For Post Body

Find  <data:post.body/> and replace it with

<div itemprop='articleBody'>

<data:post.body/>
</div>

General Cases 

In this code snippet we will cover

  1. Blog Title
  2. Post Image
  3. Author
  4. Description
<!-- Schema Mark Up by @franklinrockz -->
<b:if cond='data:blog.pageType == &quot;item&quot;'>

 <meta content='article' itemtype='http://schema.org/BlogPosting'/>

  <meta expr:content='data:blog.title' itemprop='name'/> 
  <b:if cond='data:blog.postImageThumbnailUrl'>
    <meta expr:content='data:blog.postImageThumbnailUrl' itemprop='image'/>
  </b:if>
<b:else/> 

</b:if>
<b:if cond='data:blog.url != data:blog.homepageUrl'>
<b:if cond='data:blog.pageType != &quot;item&quot;'>
<b:if cond='data:blog.metaDescription != &quot;&quot;'>
  <meta expr:content='data:blog.metaDescription' itemprop='description'/>
</b:if>
</b:if>
</b:if>
<meta expr:content='data:blog.canonicalUrl' itemprop='url'/>

<!-- Schema Mark Up -->

Voila !! We are done.
